# Kiroween Smallbiz Builder

A code-free website builder for small business owners. Fill out a form and get a complete landing page (HTML, CSS, JSON config, and deployment instructions) that you can deploy to Netlify, GitHub Pages, or Cloudflare Pages.

## What it does

- Form with presets for:
  - Coffee shop
  - Bakery
  - Barber shop
  - Food stall
  - Laundry service
  - Photographer / creator
- Generates:
  - `index.html` – landing page
  - `styles.css` – styling
  - JSON for menu/services
  - `deployment-instructions.txt` – how to deploy
